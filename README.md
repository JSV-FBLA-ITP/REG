# FBLA Virtual Pet - Competitive Programming Entry

A professional, centered-layout virtual pet simulation built with Vanilla JS.

### Rubric Requirements Met:
- **Customization**: User names pet and chooses personality via customize page.
- **Financial Responsibility**: Expense log tracks all costs of care.
- **UX/UI**: Dark Mode toggle with persistent memory.
- **Validation**: Regex used for name verification (2-12 letters).
- **Data Persistence**: Stats, money, and theme saved via LocalStorage.

### Setup
Simply open `index.html` in any modern web browser.
