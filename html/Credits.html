Technical Details

This program uses Regex Validation to ensure user input is secure and formatted correctly. Local Storage allows the user's progress and Dark Mode preferences to persist across sessions.

Cost of Care System

Designed to fulfill the FBLA 2025 topic, the in-game currency simulates the financial burden of pet ownership, requiring a balance between earning funds and spending on health/happiness.
